# Streak milestone texts — утверждённые шаблоны

Тексты milestone-сообщений для habit-tracker'а, согласованные с Эмином 2026-05-12.

**Правила tone:**
- Тёплый, поощряющий, не давить на жалость
- Эмодзи в начале (опционально, в части вариантов)
- Восклицание в конце
- `«Название привычки»` — плейсхолдер, в коде подставляется `habit.name`
- Не упоминаем юзера как «меньшинство», «один из немногих» — обзывание
- Не давим: «жалко бросать», «потеряешь стрик» — нет
- Несколько вариантов на milestone выбираются deterministic random'ом по seed `userId + scope + habitId + milestone`
- Вариант с пометкой `(только при первом достижении)` — отдельная категория. Использовать ТОЛЬКО если у юзера это первый раз когда он добивает данного milestone'а (с любой привычкой для per-habit, или впервые для overall).

**Эмодзи прогрессии (черновик, ждёт финального ОК):**
- 3 дня — 🌱 росток
- 5 дней — 🌿 побольше росток
- 10 дней — 🔥 / 🌳
- 15 дней — 🔥 / ⚡ / 🏔 (для первого раза)
- 30 дней — 🏆 / 🎉
- Для overall: 🎉 / 🔥 / ❤️ — см. план

## Per-habit milestones

### 3 дня (effect: 👍 лайк)

1. 🌱 3 дня подряд ты выполняешь привычку «Название привычки»! Она уже начинает прорастать, двигаемся дальше!
2. Привычка «Название привычки» работает 3 дня без сбоев, начало положено!
3. Серия привычки «Название привычки» уже 3 дня! Ты уже не пробуешь, а делаешь!
4. 3 дня подряд привычки «Название привычки». Мозг начинает воспринимать это как норму!)
5. Твой первый успех в привычке «Название привычки», поздравляю со стриком в 3 дня, начало положено! (только при первом достижении 3-дневного стрика)

3. Серия привычки «Название привычки» уже 5 дней! Подсознание считает это нормой!
4. 5 дней подряд привычки «Название привычки». Будни прожиты – это уже не «попробую», а «делаю»!
5. 💪 5 дней без пропусков по привычке «Название привычки»! Мозг прокладывает дорожки.
6. 🌿 5 дней подряд по привычке «Название привычки»! Привычка укрепляется, дальше пойдёт легче.

## Early-adopter rank bonus (30/60/100-day milestones)

Применяется к 30-, 60- и 100-дневным milestone'ам (per-habit 30-day И overall 30/60/100-day), приписывается отдельной строкой к выбранному варианту. Считается среди уникальных юзеров: AchievementEvent → COUNT(DISTINCT userId) WHERE milestone=X AND createdAt < this.createdAt. Если rank ≤ 10 — добавляем строку.

**Формулировка (одобрено Эмином 2026-05-12):**

> Ты {N}-й человек в трекере с таким стриком!

`{N}` подставляется как число (1, 2, ..., 10). Для N=1 можно рассмотреть отдельный shortcut "Ты первый человек..." (TBD).
