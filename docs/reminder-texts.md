# Custom reminder texts — утверждённые шаблоны

Тексты обёрток reminder'ов (утренний/вечерний/per-habit) для habit-tracker'а. Согласованы с Эмином 2026-05-12.

**Структура reminder'ов в коде:**
- **Утренний** — header (приветствие + intro к списку) + список привычек + footer (мотивашка). Кастомизируем header и footer.
- **Вечерний** — header (приветствие + intro к списку) + список привычек с inline-кнопками. **Footer'а нет.** Кастомизируем только header.
- **Per-habit** — единый шаблон «⏰ Пришло время: *эмодзи название*» + одна кнопка. Кастомизируем шаблон целиком.

**Triggers:**
- `normal` — обычное напоминание
- `near_milestone` (overlay) — близко к 5/10/15-дневному стрику
- `perfect_week_ahead` (overlay) — если сегодня закроет всё, выйдет perfect week
- `missed_1_day` (replacing) — пропустил вчера overall
- `missed_3_days` (replacing) — пропустил 3 дня overall
- `missed_7_days` (replacing) — пропал на неделю
- `recovery` (replacing) — после 30+ дней пропуска
- `habit_missed_1_day` (overlay/replacing per-habit) — конкретная привычка пропущена вчера
- `habit_missed_N_days` (overlay/replacing per-habit) — конкретная привычка пропущена N дней

**Гендерная нейтральность:** не использовать past tense глаголы с подразумеваемым гендером юзера («сделал/а», «взял/а», «включился»). Использовать present, infinitive, imperative, краткие прилагательные на согласованных существительных.

**Тире:** en-dash «–», не em-dash «—». Кавычки русские «».

Заменяет normal per-habit шаблон. Триггер: конкретная привычка пропущена N дней подряд (юзер активен в трекере, но эту привычку регулярно игнорит).

4. ⏰ Несколько дней эта привычка остаётся без галочки – держу её на всякий случай: *{эмодзи} {название}*
5. ⏰ Эта привычка стала самой пропускаемой за последние дни – вернёмся к ней? *{эмодзи} {название}*

## Overlay snippets (добавляются строкой в любую обёртку)

### near_milestone — per-habit (одобрены 2026-05-13)

Триггер: текущий стрик конкретной привычки = milestone - 1 (3, 5, 10, 15, 30). Сегодняшняя отметка по этой привычке закроет milestone.

Плейсхолдеры: `название` – имя привычки, `N` – milestone (число дней).

1. ⚡ У «название» сегодня может быть N-дневный стрик – одна отметка решает!
2. 🎯 Ещё одна отметка по «название» – и стрик дойдёт до N!
3. ⚡ «название» – одно касание до N-дневного стрика!
4. 🎯 До N-дневного стрика по «название» остался один шаг!
5. ⚡ Закроешь «название» сегодня – будет стрик N дней!
6. 🎯 Сегодняшняя отметка по «название» даст стрик в N дней!
